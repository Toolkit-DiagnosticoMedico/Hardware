# Hardware
Diseño de un Sistema Modular que permitirá medir magnitudes físicas relacionadas a la salud del paciente con la finalidad de
“subirlos la nube” para acceso de diversos servicios. Los parámetros a medir que se manejarán en ésta primera versión serán 
la temperatura, el pulso cardiaco, la presión arterial y la cantidad de la glucosa en la sangre.
